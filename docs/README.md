# Mu Documentation (GitHub mirror)

This folder contains a curated set of user-facing documentation for Mu. It is **not** the full development or implementation docs set. For the complete and authoritative docs, see the source-of-truth repository at `https://git.mills.io/prologic/mu`.

Included here:

- Getting started
- Language specification and grammar
- Roadmap and design philosophy
- Macro reference and guidance
- Standard library guidelines

If you are contributing changes, please open issues and pull requests in the source-of-truth repository.
